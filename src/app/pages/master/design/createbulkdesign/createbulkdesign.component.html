Bulk Design - Excel Import List Format

*Asterisk - In Excel Field Name Need in Excel Cells Column
Excel Field Name Description
cus_type* Customer Type(1 ->Customer 2 ->Refiner)
code_design* Code Of Design(Unique Code Apply)
is_chain Design Format -> 0 - is Not Chain , 1 - is Chain
s_hook_type Design Format ->(is_chain =1) - short type -> 1 - is Short Type , 0 - is Not Short Type
m_hook_type Design Format ->(is_chain =1) - Maximum type -> 1 - is Maximum Type , 0 - is Not Maximum Type
login_not_required Design Shows in App -> 0 - is Not Required , 1 - is Required
isnewarrival New Arrival -> 0 - Is Not Arrival , 1 - Is Arrival
expire_date New Arrival -> (1) is Expire Date
category_design* Category Of Design(Id)
subcategory_design* Subcategory Of Design (Give Category Based Subcategory Id)
collection_design* Collection Of Design(Id)
style_design* Style Of Design(Id)
theme_design* Theme Of Design(Id)
default_purity* Purity(Id)
vendor_design Karigar(Id)
vendor_designno karigar Design No
netweight_design* Design NetWeight
minweight_design* Design Minimum Weight
size Design Size
selectcustomer Design Shows In App -> 1 - Show to all, 0 - Selected Customers
design_expire Design Expire -> Set Date in Expire Of with Date , Not Set Date is Not Expire Option
created_by* Current Login(admin Id)
